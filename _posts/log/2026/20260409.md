《女神异闻录 5 皇家版》

虽然还没玩到结局，不过也先对之前的游玩做个记录。

首先是，剧情方面，开始采用倒叙回忆的方式。而回忆开始，主角因一起事故，而不得不转学到东京，并寄住在卢布朗咖啡店。在上学第一天，因误入了鸭志田的宫殿，同时觉醒人格面具。后剧情推进，在龙司与杏的协助下，攻略了第一个宫殿，同时在末尾组建了怪盗团，以此展开。后续剧情，基本就为主动或被动的触发主线剧情，后攻略宫殿，这样的节奏来。剧情内容方面，为一些常见社会现象的典型，怪盗团通过盗取（强抢）城堡中秘宝的方式，使对应人物良心发现，自我悔改，改善社会。

第一想法是，如果对坏人的惩罚，只能通过怪盗这一魔法般的手段去解决，那最后就算是世界炸了，也只能说好死。且不说怪盗的手段只能针对于个人，而最后的 boss，也是将群体意识给个体化了，才得以特攻。除此之外，也就只能做到这样了。总结为自娱自乐。不过游戏是这么设计的，那就这么玩吧。

个人剧情，支线部分，共有 23 个人，穿插在主线的个人活动时间规划推进，外加不时的邀约剧情，堪称量大管饱。模式类似 gal，通过好感触发对话，通过对话推进剧情。游玩途中，前中期基本都是自己玩，之后才看攻略选择对话选项。

人物养成，主角的五维，很少特意去刷，基本通过看书或 DVD ，外加蹭剧情奖励，基本能刷满。

战斗部分，因为皇家版有三次的、大量的、万能的、如创国之举的数万箴言，故就算选了困难难度，也基本平推。

地图探索，宫殿部分，有城堡（鸭志田）、美术馆（班目）、银行（金城）、金字塔（双叶）、宇宙基地（奥村）、赌场（新岛）、邮轮（狮童）、世界树（Boss），加还没完的丸喜研究所。体感，平均每个图耗费 3 小时探索完毕，探索流程基本为按部就班的推进，基本都能一次之内全收集。

感觉没什么好写的了。美术、音乐，都很顶级。而在最开始下载游玩的时候，因为属于是零碎时间，故感觉一般，后连续，耗费大量时间，废寝忘食的游玩，代入感就很强了。整体按照剧情，迷宫，支线，这样的流程推进。

另外，所有的可攻略女性角色中，最喜欢的是双叶，与其说是厨，不如说是把我自己映射到双叶这个角色中了。

整体游玩下来，所耗费的时间，虽说不是很有价值，但也说不上是浪费。而现在的记录与评价，也回味不深。那么就这样了。

目标，令 npc 血量为 0 ，如果是神经网络的话，参数为 {记忆 + 可感知环境 + 自我属性}，并存入记忆。那么行为是什么时候产生的，最初的记忆中有个第一因，然后过一轮（时间+记忆刷新），新的记忆中，选择最优方案执行。那么动作是在记忆中的。

```
item_枪{
	条件：基础通识，枪械通识，双手熟练；
	效果：100 范围，单点对象，伤害 100；
	单点对象{
		距离+稳定 = 命中率
	}
}
技能_抓取{
	拿起距离 1 的物品。
}
技能_位移_步行{
	前进距离 10
}
```

如果对象是人体，那么命中阈值会低，而如果对象是头，那么相应的命中阈值会高。基于随机的骰娘。

如果枪在100，而木棒在10，那么记忆轮1，计算过程为以{记忆+环境，自我属性}的广搜，得出权重图与有向图，后行动，到记忆轮2。

```

step2: 记忆搜索

```
