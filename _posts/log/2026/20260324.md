那么现在，离闭关还有 2 个半小时，接着昨天的命题，尝试 AI 工具。那么先认识下昨天搜集的几个名词。

- **GPT**：基础的大语言模型（LLM，Large Language Model）。
- **LangChain/Spring AI**，大语言模型应用开发框架，LLM + 私有数据，构建专门的 AI 工具/产品。
- **OpenClaw**，操作系统上的 AI 助理，类似手机上的小爱同学。

Google Antigravity、Cursor、Windsurf (by Codeium)、JetBrains AI (IntelliJ IDEA / PyCharm 等)、Claude Code、OpenAI Codex (Desktop App)、Kiro IDE。都是一类东西，AI 编辑器（IDE）。

理解完毕，基本上，LLM 模型的 API 调用收费，但框架或工具免费使用，而 LangChain、Spring AI 这两者属于框架，Cursor、OpenClaw 属于工具。

正常理解的，使用 AI 辅助开发，应该是指类似 Cursor 的工具，而要开发 AI 功能的业务，则需要使用 LangChain 或 Spring AI。最终开发出的东西，归属于 AI Agent？

那么，现在，继续了解 Cursor，安装，并看看与 VS Code 的区别。看了介绍，Cursor 是基于 VS Code ，集成了 AI 的功能，且在这方面使用更强大、方便的代码编辑器。想尝试下能不能做出来系统。
